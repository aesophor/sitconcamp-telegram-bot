# SITCON Camp 黑客松 - 自動圖戰機器人

**`/start`** 進行開始

**`/help`** 獲得更多資訊

inlinebot.py : bot主程式

data.json : 圖片庫

token.txt : token放置處

使用前請先在token.txt放置token

若輸入圖片名稱沒反應，可能是網址已死或沒有此圖片
